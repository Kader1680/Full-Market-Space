# Backend Section
 
## Tools used in this project
- Laravel 
- PHP 
- Mysql 

# Access the director of the backend 
```bash
cd backend
```
# install package of the project 
```bash
install composer
```
# create env file 
```bash
cp .env.example .env
```
# Run the migrations  
```bash
php artrisan migrate
```
# Link the Storage  
```bash
php artrisan storage:link
```
# Generate key  
```bash
php artrisan key:generate
```
# Start the Server locally  
```bash
php artrisan serve
```
