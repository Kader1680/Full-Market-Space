# Full-Market-Space

System Online Sell Products, user can browse all products, view by detail, place order, pay online (stripe or paypal)

## Tools used in this project
- Nextjs 
- Laravel 
- PHP 
- Mysql 
- Docker 

 

# Run the following command in your terminal to download the project locally:
```bash
git clone https://github.com/Kader1680/Full-Market-Space 
```
# Access the director of the project
```bash
cd Full-Market-Space 
```
