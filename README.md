# Full-Market-Space

System Online Sell Products, user can browse all products, view by detail, place order, pay online (stripe or paypal)


## Features of Project
- Create Account (Register), login
- Access the profile
- Browse Prodcuts
- View Product
- Filter By Category 
- Search Specific Product 
- Add to carts
- Manage the carts(edit quantiy or delete product) 
- Place order
- Manage Orders
- Track the Order Status
- Make Payment Online (Stripe or Paypal)



## Tools used in this project
- Nextjs 
- Laravel 
- PHP 
- Mysql 
- Docker
